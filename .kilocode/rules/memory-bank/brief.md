# Brief del Progetto - Qwen3 e LLaMA Factory  
  
## Panoramica  
Questo progetto riguarda l'addestramento e l'ottimizzazione del modello linguistico Qwen3 utilizzando il framework LLaMA Factory. Il Memory Bank contiene la documentazione necessaria per mantenere il contesto tra diverse sessioni di lavoro su questo progetto complesso di machine learning e modellazione linguistica.  
  
## Obiettivo  
L'obiettivo Š creare una memoria persistente che consenta a Kilo Code di:  
- Ricordare l'architettura del progetto di addestramento Qwen3  
- Conoscere le tecnologie e i framework utilizzati (LLaMA Factory, Transformers)  
- Capire gli obiettivi attuali di ottimizzazione e addestramento  
- Mantenere lo stato di avanzamento dell'addestramento  
- Ricordare le decisioni tecniche prese durante l'ottimizzazione del modello  
  
## Struttura principale  
- Cartella \`.kilocode/rules/memory-bank/\` contenente i file di contesto  
- File specifici per diversi aspetti del progetto (product, context, architecture, tech)  
  
## Stato attuale  
Il progetto Š in fase di sviluppo per l'addestramento e l'ottimizzazione del modello Qwen3. Sono disponibili diversi script di setup e addestramento, inclusi setup_and_train.py e setup_and_train_gguf.py. La struttura del progetto include il repository LLaMA Factory e i dati di addestramento.
